pvalue(mode="undirected", **params)

source code 

Compute the p-value of a given edge according the MLF significance filter.

Parameters:
  • mode - can be "directed" or "undirected".
  • w - integer weight of the edge.
  • ku - weighted degree of one end node.
  • kv - weighted degree of the other end node.
  • q - sum of all weighted degrees in graph dividd by 2.

    Other parameters are different for the directed and undirected cases. See __pvalue_directed and __pvalue_undirected for detailed description of parameters.

__pvalue_undirected(**params)

source code 

Compute the pvalue for the undirected edge null model. Use a standard binomial test from the statsmodels package.

Parameters:
  • w - weight of the undirected edge.
  • ku - total incident weight (strength) of the first vertex.
  • kv - total incident weight (strength) of the second vertex.
  • q - total incident weight of all vertices divided by two. Similar to the total number of edges in the graph.

__pvalue_directed(**params)

source code 

Compute the pvalue for the directed edge null model. Use a standard binomial test from the statsmodels package

Parameters:
  • w_uv - Weight of the directe edge.
  • ku_out - Total outgoing weight of the source vertex.
  • kv_in - Total incoming weight of the destination vertex.
  • q - Total sum of all edge weights in the graph.

prune(G, field='significance', percent=None, num_remove=None)

source code 

Remove all but the top x percent of the edges of the graph with respect to an edge attribute.

Parameters:
  • G - an igraph.Graph instance.
  • field - the edge attribute to prune with respect to.
  • percent - percentage of the edges with the highest field value to retain.
  • num_remove - number of edges to remove. Used only if percent is None.

compute_significance(G)

source code 

Compute the significance for each edge of a weighted graph according to the Marginal Likelihood Filter (MLF).

Parameters:
  • G - igraph.Graph instance. Edges must have the 'weight' attribute set. G can be directed or undirected. Each case is treated separately. For each edge a new "significance" attribute will be set.
